SnapDoctor - OpenClaw Configuration Doctor

A self-healing plugin for OpenClaw Gateway. Automatically backs up config before changes, checks health, and restores from snapshots.

Features:
- Manual backup before config changes (REQUIRED!)
- Health check to detect issues
- One-click restore from snapshots
- Auto-recovery via Heartbeat/Cron

Perfect for preventing config disasters and zero-downtime management.

Note: This plugin restores ONLY OpenClaw config files, NOT your business code.
